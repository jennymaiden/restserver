# Sistema de monitoreo de red basado en la latencia
Este RESTAPI esta creado en nodeJS es por eso que  para poder empezar a correr la aplicación es necesario descargar las instancias requeridas con el siguiente comando

npm install

# Que hace?
El API REST consiste en un conjunto de microservicios que prestan soporte a la pagina web que le tendra disponible la información de las muestras
y los diagnosticos para hacer uso de las gráficas y las tablas de resumen de una muestra
